**Build a binary file example:**

In the root directory GoGamesProject run

```
go build basic_server.go
```

**Run just one test file example:**

In GoGamesProject/dice run

```
go test
```

**Run all test files example:**

In GoGamesProject run

```
go test ./...
```

**Build local server**

```
go build basic_server.go
./basic_server
```

Paste 

```
http://localhost:8000
```

In your browser
